Analisi prima dose - Italia

ggplot(vaccini, 
       aes( x = data_somministrazione,
            y = prima_dose,
            color = fascia_anagrafica)) +
  geom_line() +
  facet_wrap( ~fascia_anagrafica)
platea_summary <- platea %>% 
  group_by(fascia_anagrafica) %>% 
  summarise(totale_popolazione = sum(totale_popolazione))

prog_vaccini <- vaccini %>%
  group_by(data_somministrazione, fascia_anagrafica) %>% 
  summarise(prima_dose = sum(prima_dose)) %>% 
  ungroup() %>% 
  group_by(fascia_anagrafica) %>% 
  mutate( prog_shot = cumsum(prima_dose)) %>% 
  left_join(platea_summary) %>% 
  mutate( prog_perc = prog_shot / totale_popolazione) %>% 
  ungroup()
  
ggplot(prog_vaccini, 
       aes( x = data_somministrazione,
            y = prog_perc*100,
            color = fascia_anagrafica)) +
  geom_line(size=1) +
  coord_cartesian(ylim = c(0,100)) +
  labs( title = "Vaccinati prima dose",
        subtitle = "Percentuale di completamento per et",
        y = NULL,
        x = NULL)

Ogni curva ha tre fasi: la prima che sale lentamente, una fase ripida dove molti di vaccinano in poco tempo, una terza parte pi piatta che corrisponde alle difficolt nel coinvolgere la porzione pi esitante per quella fascia det.

In altre parole, pi la curva ripida pi la gente rapida nelladerire alle opportunit di vaccinarsi per la loro fascia det:

  • gli 80+ sono quasi completamente vaccinati
  • i 70-79 si sono fatti vaccinare rapidamente
  • andando verso i pi giovani si nota una progressivo appiattimento della fase ripida
